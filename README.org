*** Setup

+ Install the dependencies:

#+BEGIN_SRC bash
pnpm install
#+END_SRC

## Get started

+ Start the dev server
+ This app will be available at [http://localhost:5005](http://localhost:5005).

#+BEGIN_SRC bash
pnpm dev
#+END_SRC

+ Build the app for production:

#+BEGIN_SRC bash
pnpm build
#+END_SRC

+ Preview the production build locally:

#+BEGIN_SRC bash
pnpm preview
#+END_SRC
