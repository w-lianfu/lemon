*** Setup

+ Install the dependencies:

```bash
pnpm install
```

## Get started

+ Start the dev server
+ This app will be available at [http://localhost:5005](http://localhost:5005).

```bash
pnpm dev
```

+ Build the app for production:

```bash
pnpm build
```

+ Preview the production build locally:

```bash
pnpm preview
```
